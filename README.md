# cmdutil
cmdutil is a novelty command line application written in C++ to demonstrate principles of argument parsing.


## Usage
cmdutil offers multiple submodules. You can see all submodules by using `./cmdutil` or `./cmdutil help`.
You can see help for certain modules by using `./cmdutil <submodule> help`.


## Building
```
git clone https://github.com/itsnotqwerty/cmdutil-cpp
cd cmdutil-cpp
make
```
